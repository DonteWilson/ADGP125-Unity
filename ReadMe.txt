Unity Test

What is its Purpose
----------------------------
To demonstrate my ability to use a dll as a namespace in unity and be able to 
implement code into it.

Controls
----------------------
d - Changes State to WALK
space bar - Changes State to JUMP
q - Changes State to Kick
a - Changes state to Run


How to use Unity Project
------------------------------------
Open Unity125.exe
Select Windowed and Desired Screen Resolution
Use the Controls Above.
